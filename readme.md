# 📝 To-Do List

Una aplicación simple de lista de tareas donde puedes agregar, marcar y eliminar tus pendientes diarios.

## 🚀 Funcionalidades

- ✅ Agregar tareas
- ✔️ Marcar tareas como completadas
- 🗑️ Eliminar tareas
- 📱 Responsive (si aplica)

## 🛠️ Tecnologías usadas

- HTML5
- CSS3
- JavaScript
- Bootstrap 5.3
